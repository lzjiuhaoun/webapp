# CLAUDE.md — situation 态势感知模块

## 模块使命

聚合多源数据，提供安全态势的可视化展示：总体态势、风险态势、数据流转、事件检索、健康态势和漏洞态势。

## 技术栈

- MyBatis + MySQL
- Spring Data Elasticsearch（事件检索）
- 后台线程（数据聚合）
- Netty（数据流传输）
