# INTERNAL_LOGIC.md — situation 态势感知模块

## 核心流程

### 态势数据聚合
```
risk 模块 → 风险统计数据
db-assets → 资产统计数据
device → 设备运行数据
sensdata → 敏感数据统计
            ↓
    Situation Service 聚合层
            ↓
    按维度组织 → 趋势/排名/分布
            ↓
    前端图表展示
```

### 数据流转图
- `AutoGenerateDataflowDiagramThread` 后台线程
- 根据数据源、设备、数据库的关系生成流转拓扑
- 使用 `DataFlowDottedComponentRelation` / `DataFlowSolidComponentRelation` 构建关系

### 事件检索
- 基于 Elasticsearch 的全文检索
- 支持多条件组合查询
- 返回命中记录和统计信息
