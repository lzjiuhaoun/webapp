## IM日志字段信息

## 日志标准字段：

|                  |        |        |          |                     |                       |
| ---------------- | ------ | ------ | -------- | ------------------- | --------------------- |
| **字段**           | **类型** | **说明** | **是否必填** | **示例**              | **IM****是否有该字段**      |
| id               | String | 日志ID   | 是        | 4611686027042922972 | 是                     |
| moduleSubName    | String | 功能模块   | 是        | 登录模块                | 是                     |
| ip               | String | ip地址   | 是        | 127.0.0.1           | 是                     |
| describe         | String | 日志详情   | 是        | 张三用户名密码登录成功！        | 是                     |
| operator         | String | 操作人    | 是        | 张三                  | 是                     |
| operatorId       | String | 操作人ID  | 是        | 12313136239816183   |                       |
| operatorUserName | String | 操作人姓名  | 是        | zhangsan            |                       |
| createTime       | String | 操作时间   | 是        | 1784166837778       | 是                     |
| logType          | String | 日志类型   | 是        | 登录、登出、私聊、群聊         | 是                     |
| operationResult  | String | 操作结果   | 是        | 成功                  | 是                     |
| fileName         | String | 文件名称   | 是        |                     | 否<br><br>日志类型为上传文件时有值 |
| fileUrl          | String | 文件地址   | 是        |                     | 否<br><br>日志类型为上传文件时有值 |
| participatyParty | String | 参与方名称  | 是        |                     |                       |
